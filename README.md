<p>

### Usage

To run this project use the following commands<br/>

<pre align="center">yarn install or npm install</pre>
<pre align="center">react-native run-ios</pre>

### Includes

- [x] TypeScript
- [x] React Native
- [x] Redux Saga
- [x] Tslint
- [x] Styled Components
- [x] React Navigation
